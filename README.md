# 📊 SQL Data Analysis Task (Task 3)

## 📌 Overview
This project focuses on using SQL to extract and analyze data from an ecommerce dataset.  
The objective is to practice writing SQL queries for data analysis.

---

## 🛠 Tools Used
- MySQL / DB Fiddle
- SQL

---

## 📂 Dataset
A sample ecommerce dataset was created with the following tables:
- **customers**
- **orders**

---

## 🔍 SQL Operations Performed

### ✅ Basic Queries
- SELECT statements to retrieve data

### ✅ Filtering
- WHERE clause to filter records

### ✅ Sorting
- ORDER BY to sort results

### ✅ Aggregations
- SUM() and AVG() functions
- GROUP BY for grouping data

### ✅ Joins
- INNER JOIN
- LEFT JOIN

### ✅ Subqueries
- Used nested queries for analysis

### ✅ HAVING Clause
- Filtered grouped data

### ✅ Views
- Created a view for customer spending analysis
